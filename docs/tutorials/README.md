## Production Notes

### Visual Style
- Use screen recordings with cursor highlights
- Add callout boxes for important features
- Use zoom effects for small UI elements
- Include captions for accessibility

### Audio
- Clear, professional narration
- Background music (subtle, non-distracting)
- Sound effects for actions (optional)

### Branding
- Include company logo in intro/outro
- Consistent color scheme with the app (blues, purples for AI features)

## Recording Checklist

Before recording:
- [ ] Use sample data (not real inspection data)
- [ ] Clear browser cache for clean state
- [ ] Disable notifications
- [ ] Use consistent window size (1920x1080 recommended)
- [ ] Prepare demo files with varied data
