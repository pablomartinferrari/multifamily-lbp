### Scene 2: Export Options Overview (0:20 - 0:45)

**[VISUAL]** Show diagram of export points

**[NARRATION]**

### Scene 3: Exporting from Data Review Grid (0:45 - 1:30)

**[VISUAL]** Navigate to Data Review Grid

**[NARRATION]**
> "The Data Review Grid is where you see all your readings after processing."

**[VISUAL]** Click Export to Excel button
> "Click **Export to Excel** to download a complete spreadsheet."

**[VISUAL]** Show Excel file opening with columns highlighted
> "The export includes ALL columns:
> - Reading ID
> - Location and Room
> - Original Component and Substrate
> - Normalized Component and Substrate  
> - Lead Content value
> - Positive/Negative result
> - Color (if captured)
>
> This is your raw data export—perfect for detailed analysis or archiving."

### Scene 4: Exporting Summary Reports (1:30 - 2:15)

**[VISUAL]** Navigate to Summary Report

**[NARRATION]**
> "After generating a summary, you can export the classification results."

**[VISUAL]** Show multi-sheet Excel workbook
> "This creates a workbook with multiple sheets:"

**[VISUAL]** Show each sheet:
> "**Average Components**: Components with 40+ readings, showing the 2.5% threshold analysis.
>

>
> **Non-Uniform Components**: Mixed-result components with detailed breakdowns."

**[VISUAL]** Show Component and Substrate columns
> "Each sheet has separate Component and Substrate columns, plus all the statistics: positive counts, negative counts, percentages, and total readings."

### Scene 5: Exporting from All Shots Report (2:15 - 2:50)

## Key Points to Emphasize

1. Three export points: Data Review, Summary, All Shots
2. Summary exports are multi-sheet workbooks
3. Both Excel and CSV formats available
4. Component and Substrate are separate columns
5. Exports include job number in filename
